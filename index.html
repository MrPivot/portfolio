  • Primary Role: Physics Programmer
  • Type: Ludum Dare - Game Jam
  • Tech: Unreal Engine 4 / Blueprints
  • Platform: PC

Custom Moba Engine
February 2017 - April 2017

This was a second year project during which we made a custom engine for Mobas. I mainly worked on the OpenGL renderer, GLSL shaders, shadowmapping and frustum culling using a quadtree.

  • Type: School Project
  • Tech: Custom Engine / C++ / OpenGL / GLSL
  • Platform: PC
