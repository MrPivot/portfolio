Project Ragnarok
September 2017 - Present

Project Ragnarok is a turn-based tactical RPG where you lead a team of mechs, customized by you, with the goal of destroying your opponent and capturing their base. This project is being built with a team of +/- 20 people during the entire third year of igad.

My main contributions to this project are:
- The procedural generation of the level.
- Grid construction and initialization.

  • Primary Role: Procedural Programmer
  • Type: School / Client Project
  • Tech: Unreal Engine 4
  • Platform: PC / PS4

Beat Batten
May 2017 - June 2017

Beat Batten is a turn-based card game featuring networked player vs player gameplay. Using your guardians you try to either beat your opponent (Batten) or reach the door at the other side of the map. This game was made for the Beat Batten foundation, which aims to find a cure for Batten disease. This project was implemented using Entitas, a data oriented Entity Component System.

My main contributions to this project where:
- Camera controls.
- Implementation of animation.
- Parts of the cards and combat.

  • Primary Role: Gameplay Programmer
  • Type: School / Client Project
  • Tech: Unity / Custom Networking Library / Entitas
  • Platform: Mobile

- Other Projects -

FishXBrawl
April 2018

This was a game, made in a small team, for Ludum Dare 41. Goal of the game is to fish up your weapon and then use it to slap your friends and / or foes out of the map. I mainly worked on the implementation of the physics.

Unreal Grid Editing Module
February 2018 - March 2018

This was a small grid editing tool that I implemented for Project Ragnarok. My main goal implementing this was exploring to what extent I could manage to extend Unreal Editor.

  • Type: School / Personal Project

  • Platform: PC

Custom Moba Engine
February 2017 - April 2017

This was a second year project during which we made a custom engine for Mobas. I mainly worked on the OpenGL renderer, GLSL shaders, shadowmapping and frustum culling using a quadtree.

  • Primary Role: Graphics Programmer
  • Type: School Project
  • Tech: Custom
  • Platform: PC

Neon Death Cube
December 2016

This was a game, made in a small team, for Ludum Dare 37. It is a multiplayer first person shooter situated inside a cube. This was my first time working with Unreal Engine, I mainly worked on gameplay and shifting gravity.

  • Primary Role: Gameplay Programmer
