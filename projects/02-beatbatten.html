- Description -

Beat Batten is a turn-based card game featuring networked player vs player gameplay. Using your guardians you try to either beat your opponent (Batten) or reach the door at the other side of the map. This game was made for the Beat Batten foundation, which aims to find a cure for Batten disease. This project was implemented using Entitas, a data oriented Entity Component System.

  • Status: Completed
  • Primary Role: Gameplay Programmer
  • Type: School / Client Project
  • Tech: Unity / C# / Custom Networking Library / Entitas
  • Platform: Mobile
  • Client: Beat Batten Foundation
